Stock Trend Snapshot
Web base application that will retrieve some basic information on a stock and show the stock's movements with arrows.



Installation Frontend
mkdir stocktrend
npx create-react-app frontend
npm i axios react-redux redux redux-thunk react-router-dom
npm i react-number-format

npm start // will start react page

Installation Backend
mkdir backend
npm init
npm install node.js
npm install express axios
npm install i cors
npm install mongoose mongodb
npm i dotenv

nodemon // will start node server
mongod // will start mongo database

Will make API get request to Alphavantage and to IEX.

Login verification will be done to user database at localhost5000
